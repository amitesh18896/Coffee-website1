# Coffee Shop
A coffee shop website demo.

[View Live](https://local-coffee-demo.netlify.app/)

## Technologies Used
- CSS3
- HTML5
- JavaScript

## Features
- Dark/Light theme switch
- CSS grid
- Testimonials slider (JavaScript)
- embedded google maps
- Responsive design
